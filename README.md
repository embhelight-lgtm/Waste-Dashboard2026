# Waste Performance Dashboard 2026

Dashboard untuk visualisasi data waste proses produksi.

## Fitur
- **Visualisasi Trend**: Trend mingguan dan bulanan waste percentage.
- **Analisis Customer**: Detail waste per customer dan per item.
- **Fulfillment Rate**: Monitoring tingkat pemenuhan order.
- **Filter Dinamis**: Filter berdasarkan Tahun, Bulan, dan Nama Customer.

## Cara Update Data
1. Jalankan script `extract_waste.ps1` melalui PowerShell atau klik `RUN_DASHBOARD.bat`.
2. File `waste_data.js` akan diperbarui secara otomatis.
3. Commit dan Push perubahan ke GitHub (jika menggunakan Git) atau unggah ulang file `waste_data.js` ke repository.

## Teknologi
- HTML5 & CSS3 (Vanilla)
- JavaScript (Vanilla)
- [Chart.js](https://www.chartjs.org/)
